# **Welcome to our Balanced Plate!**

### Many people find themselves struggling with making a balanced, healthy plate and with our application, that's possible! We make it easy to choose items from specific food groups and then view them all in a compiled list.

**To run our application:**

1. Open up your code editor of choice and clone the repo. Then, install all dependences:
   ```
   git clone [REPO_URL]
   npm install
   npm install react-icons
   ```

3. Run the application:
   ```
   npm start
   ```
